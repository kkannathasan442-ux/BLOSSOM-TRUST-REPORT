# Blossom Trust Student Reporting Portal

Centralized student reporting system and admin dashboard designed for Blossom Trust.

## Project Structure
- `/frontend` - React & Vite application containing the Student Portal and Admin Dashboard.
- `/backend` - Express.js REST API, SQLite database, and integration handlers (Google Sheets, PDF/Excel generation).

## Setup
### Backend
1. Navigate to `/backend`.
2. Run `npm install` to install dependencies.
3. Run `npm start` to initialize the database and start the server on port 5000.

### Frontend
1. Navigate to `/frontend`.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the frontend application.
